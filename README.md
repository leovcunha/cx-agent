# Customer Service Agent Platform

A plug-and-play full-stack Customer Service Agent platform. This project allows businesses to configure an AI support agent and upload their Standard Operating Procedures (SOPs) via an Admin Dashboard. The agent dynamically retrieves and reasons over these SOPs to provide Tier-1 support.
